# Tech Stack Foundation - CAD

This directory contains the OpenSCAD files for the 3D printed foundation.

## Files
- `foundation.scad`: Main design file.

## Usage
1. Install [OpenSCAD](https://openscad.org/).
2. Open `foundation.scad`.
3. Press `F5` to preview.
4. Press `F6` to render.
5. File -> Export -> Export as STL.

## Print Settings
- **Material**: PETG or ABS recommended for thermal stability.
- **Infill**: 20%+, Grid or Gyroid.
- **Orientation**: Flat on bed.
